쿼터

쿼터 실습

설치 직후로 초기화하자.

새로운 디스크를 추가해서 쿼터 전용 디스크로 하자
사용자의 홈 디렉토리는 별도의 디스크로 하는것이 효과적이다. 그리고 사용자별 쿼터도 지정해보자

1. 20기가짜리 하나 디스크 달자
2. 유저 두명 추가하자
3. 아까 추가한 디스크를 쿼터용으로 설정하자
4. 쿼터디비생성

명령어들에 대해 관심있으면 책을보자

5. 개인별 쿼터 설정

블록(block)이 현재 사용하는 키로바이트 수

소프트제한(soft)- 넘어갈수 있지만 기간이 있음
하드제한(hard) -완전히 못넘어감
